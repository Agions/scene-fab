# 架构概览

## 系统架构

Voxplore 采用**分层模块化**架构，核心逻辑与 UI 完全解耦。

```
┌─────────────────────────────────────────────────┐
│                   Voxplore App                    │
├─────────────────────────────────────────────────┤
│  UI Layer (PySide6/Qt)                           │
│  ┌──────────┐ ┌──────────┐ ┌──────────────┐    │
│  │MainWindow│ │HomePage  │ │ Step Pages   │    │
│  └──────────┘ └──────────┘ └──────────────┘    │
├─────────────────────────────────────────────────┤
│  Business Logic Layer                             │
│  ┌──────────┐ ┌──────────┐ ┌──────────────┐    │
│  │ VideoMgr │ │Analysis  │ │ ExportEngine │    │
│  └──────────┘ └──────────┘ └──────────────┘    │
├─────────────────────────────────────────────────┤
│  AI Service Layer                                │
│  ┌──────────┐ ┌──────────┐ ┌──────────────┐    │
│  │DeepSeek  │ │ Qwen VL  │ │ Edge TTS     │    │
│  └──────────┘ └──────────┘ └──────────────┘    │
├─────────────────────────────────────────────────┤
│  Platform Layer (FFmpeg / System)                │
└─────────────────────────────────────────────────┘
```

## 核心模块

| 模块 | 路径 | 职责 |
|------|------|------|
| `app.ui` | `app/ui/` | PySide6 桌面 UI 组件 |
| `app.core` | `app/core/` | 核心业务逻辑 |
| `app.services` | `app/services/` | AI 服务封装 |
| `app.utils` | `app/utils/` | 工具函数 |
| `app/config` | `app/config/` | 配置管理 |

## 数据流

```
视频输入 → 预处理 → 帧提取 → Qwen VL 分析
    ↓
声纹提取 → 混合分组（视觉 0.7 + 声纹 0.3）
    ↓
情感评分 → 叙事排序 → 片段选择
    ↓
DeepSeek 生成解说稿 → TTS 配音合成
    ↓
FFmpeg 合并 → 输出 MP4 / 剪映草稿
```

## 配置管理

使用 `pydantic-settings` 进行配置校验：

- 环境变量优先
- `.env` 文件支持
- 敏感信息不写入代码
