# 功能矩阵

SceneFab 是面向影视/短剧第一人称解说的生产工具，将素材理解、脚本生成、配音字幕和平台导出串成标准化流程。以下按功能模块列出当前能力与开发状态。

## 功能状态说明

| 标记 | 含义 |
| --- | --- |
| 已发布 | 功能完整可用，经过测试验证 |
| 开发中 | 核心逻辑已实现，正在完善边界场景 |
| 计划中 | 已列入开发计划，尚未开始实现 |

## 视频理解

| 功能模块 | 能力 | 状态 |
| --- | --- | --- |
| 场景分析 | 自动识别场景切换、提取关键帧和画面摘要 | 已发布 |
| 人物识别 | 识别画面中的人物、推断角色关系 | 已发布 |
| 情绪峰值 | 检测冲突、反转、高潮等情绪节点 | 已发布 |
| 桥段检测 | 识别短剧常见桥段（打脸、逆袭、误会等） | 已发布 |
| 剧情图谱 | 构建 StoryGraph，追踪人物和剧情线 | 已发布 |
| 多场景长视频 | 支持电影/剧集级别的长视频分段理解 | 已发布 |
| 音频线索提取 | 从原片音频中提取对白和声音事件 | 开发中 |

| --- | --- | --- |

## 适用场景

| 场景 | 推荐用法 |
| --- | --- |
| 短剧单集解说 | 录入题材、爽点、人物关系和集数上下文，优先生成 30-90 秒竖屏稿 |
| 短剧整季批量 | 先统一标签和关系设定，再按集生成脚本、配音和导出记录 |
| 电影/剧集片段解说 | 先做场景拆分和情绪峰值筛选，再生成第一人称视角脚本 |
| 剪映继续精剪 | 导出剪映草稿，保留字幕、配音和时间轴供人工二次处理 |

## 不适用场景

| 场景 | 原因 |
| --- | --- |
| 逐帧精修调色 | SceneFab 只负责生产链路，不替代专业调色软件 |
| 复杂多轨包装 | 多机位、复杂特效和商业包装建议进入专业剪辑工具继续处理 |
| 无版权素材搬运 | 项目要求素材来源可追溯，最终内容合规由使用者负责 |
