## 侧边栏

| 图标 | 功能 | 快捷键 |
|------|------|--------|
| 🏠 主页 | 首页视图 | `Alt+1` |
| ＋ 创作 | 新建/继续创作 | `Alt+2` |
| ☰ 项目 | 项目管理 | `Alt+3` |
| ⚙️ 设置 | 配置选项 | `Alt+4` |

## 创作流程

```
[上传] → [AI 拆条] → [解说生成] → [导出]
  ①         ②           ③          ④
```

### ① 上传

拖拽或选择视频文件夹，支持 mp4 / mov / avi / webm

### ② AI 拆条

等待 Qwen VL 分析，自动按语义切分场景

### ③ 解说生成

预览片段，调整解说风格和文案，生成配音

### ④ 导出

选择格式和质量，导出最终视频
