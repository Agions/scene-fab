# 界面介绍

SceneFab 提供直观的桌面界面，将复杂的 AI 视频制作流程分解为清晰的步骤。本页介绍界面布局和主要功能区域。

## 界面布局

```text
┌─────────────────────────────────────────────────────┐
│  SceneFab                              [主题] [导出]  │  ← 顶部栏
├────┬────────────────────────────────────────────────┤
│ 📋 │                                                │
│ ⚙ │         主内容区域                              │
│ 🎬 │        （根据选中菜单切换）                      │
│ 📁 │                                                │
│ 🖼 │                                                │
│ 📤 │                                                │
├────┴────────────────────────────────────────────────┤
│ ● 就绪                                       v2.4.0 │  ← 状态栏
└─────────────────────────────────────────────────────┘
```

## 顶部栏

| 操作 | 功能 |
|------|------|
| Logo | 返回工作台首页 |
| 主题切换 | 浅色 / 深色 / 系统跟随（v2.4.0 新增） |
| 导出 | 跳转到导出发布页面 |

## 侧边栏

| 入口 | 图标 | 功能 |
|------|------|------|
| 工作台 | 📋 | 查看生产状态、最近资产和默认交付参数 |
| 创作流程 | 🎬 | 素材导入、场景拆分、脚本、配音字幕和导出队列 |
| 项目资产 | 📁 | 管理视频、音频、字幕和导出文件 |
| AI 配置 | 🖼 | 配置 AI 服务商（DeepSeek / Qwen / Edge-TTS） |
| 系统设置 | ⚙ | 配置模型、密钥、导出预设、工作目录和外观主题 |

## 工作台

工作台是 SceneFab 的首页，提供：

- **项目状态总览**：进行中 / 已完成 / 待处理
- **最近资产**：最近编辑的项目和导出文件
- **快速操作**：新建项目、导入素材、打开最近
- **统计信息**：累计生成时长、节省时间

## 创作流程

创作流程页面将视频制作分解为 5 个标准步骤：

| 步骤 | 操作 | 说明 |
|------|------|------|
| ① 素材导入 | 选择影视片段并记录来源 | 支持 mp4 / mov / avi / webm |
| ② 场景拆分 | AI 视频分析 | 识别人物、冲突和关键转折 |
| ③ 脚本生成 | 第一人称视角组织叙事 | Hook、桥段和节奏统一评估 |
| ④ 配音字幕 | 合成音频并校准字幕 | 字幕与配音偏差目标小于 50ms |
| ⑤ 导出发布 | 选择平台参数和质量 | MP4 / 剪映草稿 JSON |

### 步骤导航

每个步骤页面提供：

- **进度指示器**：当前步骤 / 总步骤
- **输入预览**：上一步的产出
- **配置选项**：当前步骤的可调参数
- **运行按钮**：开始处理
- **结果预览**：处理完成后的产出

## 项目资产

项目资产页面管理所有项目文件：

| 资产类型 | 说明 | 操作 |
|----------|------|------|
| 视频源文件 | 原始视频素材 | 预览 / 删除 / 重命名 |
| 配音音频 | TTS 生成的配音 | 播放 / 重新生成 / 下载 |
| 字幕文件 | ASS / SRT 格式 | 预览 / 编辑 / 下载 |
| 导出文件 | MP4 / 剪映草稿 | 预览 / 下载 / 删除 |
| 脚本文件 | 解说稿原文 | 预览 / 编辑 / 下载 |

## AI 配置

AI 配置页面集中管理所有 AI 服务：

- **服务商启用**：勾选启用的服务商
- **API Key 管理**：安全存储（加密 + keyring）
- **模型选择**：为每个服务商选择默认模型
- **测试连接**：验证 API Key 有效性

详见 [AI 配置](/guide/ai-configuration)。

## 系统设置

系统设置页面管理应用级配置：

| 设置项 | 说明 |
|--------|------|
| 缓存目录 | 视频处理缓存位置（默认 `~/.cache/scenefab`） |
| 工作目录 | 项目文件存储位置 |
| 导出预设 | 默认导出参数 |
| 外观主题 | 浅色 / 深色 / 系统跟随 |
| 日志级别 | DEBUG / INFO / WARNING / ERROR |
| 自动更新 | 检查新版本 |

## 状态栏

状态栏显示当前应用状态：

| 状态 | 含义 |
|------|------|
| ● 就绪 | 应用空闲，等待用户操作 |
| ● 处理中 | 正在执行任务，显示进度条 |
| ● 错误 | 最近一次操作失败，查看日志 |
| ● 更新可用 | 检测到新版本，点击查看 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 新建项目 |
| `Ctrl/Cmd + O` | 打开项目 |
| `Ctrl/Cmd + S` | 保存项目 |
| `Ctrl/Cmd + E` | 导出 |
| `Ctrl/Cmd + ,` | 打开设置 |
| `Ctrl/Cmd + Q` | 退出应用 |
| `F1` | 查看帮助 |
| `F5` | 刷新当前页面 |

## 主题切换

v2.4.0 支持运行时主题切换：

- **浅色模式**：默认主题，适合白天使用
- **深色模式**：低光环境使用，减少眼睛疲劳
- **系统跟随**：跟随操作系统设置自动切换

切换路径：设置 → 外观主题 → 选择模式

## 无障碍特性

- 键盘导航：所有功能可通过键盘操作
- 屏幕阅读器：关键元素有 ARIA 标签
- 高对比度：支持高对比度模式
- 字体缩放：支持自定义字体大小
