# 界面介绍

## 界面布局

```text
┌─────────────────────────────────────────────────────┐
│  工作台                                      导出    │  ← 顶部栏
├────┬────────────────────────────────────────────────┤
│ 工 │                                                │
│ 作 │                                                │
│ 台 │         主内容区域                              │
│ 流 │                                                │
│ 程 │                                                │
│ 资 │                                                │
│ 产 │                                                │
│ 设 │                                                │
│ 置 │                                                │
├────┴────────────────────────────────────────────────┤
│ 当前: 工作台                                      │  ← 状态栏
└─────────────────────────────────────────────────────┘
```

## 侧边栏

| 入口 | 功能 |
|------|------|
| 工作台 | 查看生产状态、最近资产和默认交付参数 |
| 创作流程 | 素材导入、场景拆分、脚本、配音字幕和导出队列 |
| 项目资产 | 管理视频、音频、字幕和导出文件 |
| 系统设置 | 配置模型、密钥、导出预设和工作目录 |

## 顶部栏

| 操作 | 功能 |
|------|------|
| 导出 | 跳转到创作流程，完成脚本、配音、字幕和导出检查 |

## 创作流程

| 步骤 | 操作 | 说明 |
|------|------|------|
| ① 素材导入 | 选择影视片段并记录来源 | 支持 mp4 / mov / avi / webm |
| ② 场景拆分 | AI 视频分析 | 识别人物、冲突和关键转折 |
| ③ 脚本生成 | 第一人称视角组织叙事 | Hook、桥段和节奏统一评估 |
| ④ 配音字幕 | 合成音频并校准字幕 | 字幕与配音偏差目标小于 50ms |
| ⑤ 导出发布 | 选择平台参数和质量 | MP4 / 剪映草稿 JSON |
