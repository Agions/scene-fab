# 界面介绍

## 主界面布局

```
┌─────────────────────────────────────────────────────┐
│  [Logo] SceneFab                    [🔍] [⚙️] [?]  │  ← 顶部栏
├────┬────────────────────────────────────────────────┤
│ 主 │                                                │
│ 页 │         主内容区域                              │
│ 页 │                                                │
│ 创 │                                                │
│ 作 │                                                │
│ 项 │                                                │
│ 目 │                                                │
├────┴────────────────────────────────────────────────┤
│ 就绪                                              │  ← 状态栏
└─────────────────────────────────────────────────────┘
```

## 侧边栏

| 图标 | 功能 | 快捷键 |
|------|------|--------|
| 🏠 主页 | 首页视图 | `Alt+1` |
| ＋ 创作 | 新建/继续创作 | `Alt+2` |
| ☰ 项目 | 项目管理 | `Alt+3` |
| ⚙️ 设置 | 配置选项 | `Alt+4` |

## 创作流程

| 步骤 | 操作 | 说明 |
|------|------|------|
| ① 上传 | 拖拽或选择文件夹 | 支持 mp4 / mov / avi / webm |
| ② AI 拆条 | Qwen VL 分析 | 按语义自动切分场景 |
| ③ 解说生成 | 选择风格，生成文案 | 7 种情感风格可选 |
| ④ 导出 | 选择格式和质量 | MP4 / 剪映草稿 JSON |
