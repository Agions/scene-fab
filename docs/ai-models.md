# AI 模型

SceneFab 的模型层服务于第一人称影视/短剧解说生产。模型选择的优先级是：中文叙事稳定性、成本可控、长上下文可用、失败可降级。

## 模型分层

| 层级 | 主要职责 | 推荐模型 |
| --- | --- | --- |
| 视频理解 | 分析画面、人物、动作、场景和冲突 | Qwen3.7 / Gemini 3.5 Flash |
| 解说生成 | 生成第一人称脚本、Hook、桥段和结尾钩子 | DeepSeek-V4 |
| 语音识别 | 提取原片对白和声音线索 | SenseVoice / Whisper |
| 配音合成 | 生成解说音频和时间戳 | Edge-TTS / F5-TTS |

## 视频理解

| 模型 | 适用场景 | 说明 |
| --- | --- | --- |
| Qwen3.7 | 中文短剧、影视片段、人物关系识别 | 默认优先，适合中文语境 |
| Gemini 3.5 Flash | 成本敏感、多模态补充分析 | 可作为备用视觉理解服务 |
| 本地抽帧规则 | API 不可用时 | 可降级生成基础场景片段 |

## 解说生成

| 模型 | 适用场景 | 说明 |
| --- | --- | --- |
| DeepSeek-V4 | 中文第一人称解说、短剧脚本、Hook 改写 | 默认主力模型 |
| Claude / Gemini / GLM | 长上下文或备用生成 | 适合在主模型不可用时切换 |

脚本生成时不会只传入场景描述，还会传入题材标签、人物关系、前情摘要、桥段和平台字数约束。

## ASR 与 TTS

| 类型 | 推荐 | 说明 |
| --- | --- | --- |
| ASR | SenseVoice / Whisper | 用于原片对白、说话人和声音线索 |
| TTS | Edge-TTS | 默认配音，低门槛、多音色 |
| TTS | F5-TTS | 适合音色克隆和更稳定的人设声线 |

## 选择建议

| 目标 | 推荐组合 |
| --- | --- |
| 快速出片 | Qwen3.7 + DeepSeek-V4 + Edge-TTS |
| 成本优先 | 基础场景规则 + DeepSeek-V4 + Edge-TTS |
| 音色一致 | Qwen3.7 + DeepSeek-V4 + F5-TTS |
| 长片理解 | Qwen3.7 / Gemini 3.5 Flash + StoryGraph + DeepSeek-V4 |
